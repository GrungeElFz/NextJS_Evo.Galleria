## Running Locally

Clone this repository :

```bash
> git clone git@github.com:GrungeElFz/NextJS_Evo.Galleria.git

> cd NextJS_Evo.Galleria

> code .

> npm i

> npm run dev
```

> - Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
